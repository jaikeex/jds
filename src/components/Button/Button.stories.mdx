## Button

The button component comes in three appearance variants:

- Filled (default)
- Outlined
- Subtle

<br />

### Filled button

The default appearance. Should be used on primary and emphasized actions within the application.

<Canvas>
  <Story id="design-system-button--filled" />
</Canvas>

### Outlined button

Less pronounced appeearance. Commonly used alongside primary buttons to handle secondary actions, e.g. "Cancel" action in forms, etc.

<Canvas>
  <Story id="design-system-button--outlined" />
</Canvas>

### Subtle button

Used for low importance actions. Using subtle button helps maintain focus on the surrounding content, typically in cards and dialogs.

<Canvas>
  <Story id="design-system-button--subtle" />
</Canvas>

### Click handling

All buttons accept the `onClick(event: React.MouseEvent<HtmlButtonElement>)` function as a prop.

```jsx
<Button onClick={() => console.log('Button clicked!')}>Button</Button>
```

### Sizes

The `size` prop can be used to create larger or smaller buttons.

  <Story id="design-system-button--sizes" />

### Colors

Buttons support all available color theme variants.

  <Story id="design-system-button--colors" />

### Ripple effect

The ripple effect is enabled by default on all buttons. To disable it, set the `disableRippleEffect` prop.
