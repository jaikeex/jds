### Handling input changes

The switch component accepts the `onClick(event: React.ChangeEvent<HtmlInputElement>)` function as a prop, which is used to handle changes of its state. Because the switch is based on a native html checkbox input, the checked/unchecked state can be accessed inside the `onClick()` handler as `event.target.checked`.
