## Pagination

The pagination component allows users to navigate through pages of content. It comes with configurable appearance, color, and size options.

Pagination components support all available color theme variants.

### Pagination range

The `siblingCount` prop can be used to configure the number of siblings displayed around the active page.

<Canvas>
  <Story id="design-system-pagination--sibling-count" />
</Canvas>

### Page change handling

Pagination components accept an onChange function prop to handle page changes.

```jsx
<Pagination onChange={(page) => console.log(`Page changed to: ${page}`)} />
```
