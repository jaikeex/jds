Checkbox supports all available color theme variants.

<Canvas>
  <Story id="design-system-checkbox--colors" />

### Label

The string passed in a `label` prop will be used as an input label for the given checkbox.

<Canvas>
  <Story id="design-system-checkbox--label" />
</Canvas>

### Label positions

The `labelPosition` prop controls the position at which the label will be displayed .

<Canvas>
  <Story id="design-system-checkbox--label-positions" />
</Canvas>

### Custom icon

Custom checkbox appearance can be set by passing a `icon` and `iconChecked` props, which will be used as a default and checked state, respectively.

<Canvas>
  <Story id="design-system-checkbox--icon" />
</Canvas>

### Controlled checkbox group

By manually setting the `checked` prop of an individual checkboxes a controlled group can be created. In the example below, the controlling checkbox is set to indeterminate if the values of group members are not aligned, and can itself be used to control their state.

<Canvas>
  <Story id="design-system-checkbox--checkbox-group" />
</Canvas>

### Handling input changes

Every checkbox accepts the `onClick(event: React.ChangeEvent<HtmlInputElement>)` function as a prop.
